# 今！やることリスト

締切までにやりたいことを登録しておいて、**時間が空いたときに「今やるとよさそうなこと」を提案してくれる**タスク管理アプリです。

「やることは分かっているのに、30分空いた瞬間に何をすればいいか思い出せない」を解消するために作りました。

## できること

- **登録** — やること・締切・「1ページ何分 × 何ページ」のような見積もりを入れて記録
- **一覧** — カレンダーに締切を並べ、その日の残り作業時間もひと目で分かる
- **提案** — 空き時間（例：30分）と気分（だるい／ふつう／やる気ある）を選ぶと、今やるとよさそうな順に並べて理由つきで出す
- **タイマー** — 提案から直接タイマーを開始し、終わったら「2ページ進んだ」を記録
- **バックアップ** — JSONで書き出し・読み込み
- オフラインでも動く PWA。ホーム画面に追加すればアプリのように使えます

## 提案のしくみ

3つの観点をそれぞれ100点満点にして、重みをつけて合計しています（`js/suggest.js`）。

| 観点 | 重み | 見ているもの |
| --- | --- | --- |
| 締切 | 0.38 | 期限までの残り日数と、「1日あたり何分やらないと終わらないか」という切迫感 |
| 時間 | 0.30 | その空き時間に収まるか、無駄なく使い切れるか（終わり切るなら加点） |
| 気分 | 0.32 | 今の集中力とタスクが要求する集中度、選んだ「やりたい種類」との一致 |

そのため、**大きなタスクは「ページ単位」で登録するほど提案の精度が上がります**。15分しか空いていなくても「3ページ進める」と提案できるからです。

## データの扱い

タスクは**使っている端末のブラウザ（localStorage）にだけ**保存されます。サーバーには何も送信していないので、同じURLを他の人が開いても、その人には自分専用の空のリストが表示されます。ログインは不要です。

その代わり、ブラウザのデータを消すとタスクも消えます。大事なものは設定画面からバックアップしてください。

## 手元で動かす

ES モジュールと Service Worker は `file://` では動かないので、簡易サーバーを使ってください。

```bash
node scripts/serve.mjs      # → http://localhost:5173
```

アイコンPNGを作り直したいときは:

```bash
node scripts/make-icons.mjs
```

ビルドは不要です。必要なのは Node だけ（サーバーとアイコン生成に使うだけで、アプリ本体は素の HTML/CSS/JS）。

## GitHub Pages で公開する

1. GitHub で空のリポジトリを作る（例：`now-todo`）
2. このフォルダから push する

   ```bash
   git remote add origin https://github.com/<ユーザー名>/now-todo.git
   git push -u origin main
   ```

3. リポジトリの **Settings → Pages** を開き、Source を **Deploy from a branch**、Branch を **main / (root)** にして Save
4. 数分後に `https://<ユーザー名>.github.io/now-todo/` で開けます

パスはすべて相対パスなので、サブディレクトリ配信のままで動きます。スマホで開いて「ホーム画面に追加」すればインストール完了です。

### 更新したとき

Service Worker がアプリ本体をキャッシュしているので、内容を書き換えたら `sw.js` の `VERSION` を上げてください。利用者は次回起動時に新しい版に入れ替わります。

## ファイル構成

```
index.html              画面の骨組み
styles.css              見た目（ダークモード対応）
js/store.js             データの保存・読み書き（localStorage）
js/suggest.js           提案のスコアリング
js/calendar.js          カレンダー描画
js/main.js              画面の組み立てと操作
sw.js                   オフライン用 Service Worker
manifest.webmanifest    PWA の設定
scripts/serve.mjs       確認用の簡易サーバー
scripts/make-icons.mjs  アイコンPNGの生成
```
